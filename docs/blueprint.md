# **App Name**: BookBridge

## Core Features:

- Book Management: Add, delete, and modify book information (title, author, category, etc.).
- Member Management: Register, delete, and modify member information (email, etc.).
- Book Status Tracking: Track the status of each book (available, reserved, borrowed).
- Book Search: Search for books by category, title, author, or field.
- Email Login: Allow users to log in with their email address.
- Persistent Login: Remember login information for automatic login on subsequent visits.
- Rental History Report: Generate monthly and yearly reports of rental history for all members. Store the logs in a Cloud SQL database

## Style Guidelines:

- Primary color: Deep purple (#673AB7) to convey a sense of knowledge and sophistication.
- Background color: Light grey (#F0F0F0) for a clean and modern look.
- Accent color: Soft lavender (#D1C4E9) for interactive elements.
- Body and headline font: 'Alegreya', a serif with a contemporary feel, suitable for both headlines and body text.
- Use simple, consistent icons for book categories and actions (borrow, return, etc.).
- Clean and organized layout with clear sections for book listings, member management, and reports.
- Subtle animations on page transitions and user interactions to enhance the user experience.
